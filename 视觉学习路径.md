## 3. Linux (Ubuntu)
支持的安装/使用方式：
1. 双系统
   - 优点：驱动最完备，无明显 bug
   - 缺点：不同系统操作习惯差异大；Linux 下的 Clash Verge 已停止维护，无法使用，需用 V2ray
2. WSL（Windows 子系统）
   - 优点：与 Windows 深度集成；VS Code 联动简单；GUI 可在 Windows 下便捷调出
   - 缺点：虚拟显卡驱动较老，仅支持 OpenGL 3.3，无法强制使用 OpenGL 4.X
3. VMware（虚拟机）
   - 缺点：属于重型工具，转译层损耗较大
   - 优点：虚拟显卡驱动较新，支持 OpenGL 4.X；可配置 NAT 或桥接模式，与主机共享网络设置，无需手动配置代理

1. Linux 的指令：命令行各类指令（重点学习）
