## <center>CPU 版 YOLO v8 环境配置

### 第一步：安装 YOLO v8 (Ultralytics 库)

确保你还在 `(yolo)` 环境下，复制并运行这行命令（同样使用清华源加速）：

```bash
pip install ultralytics -i https://pypi.tuna.tsinghua.edu.cn/simple
```

*这个库会自动帮你下载 OpenCV（处理图像用的）、Pandas（处理数据用的）等所有必要的依赖包*

---

### 第二步：编写程序进行验证


1. 新建一个文件夹，命名为 `yolo_study`
2. 在这个文件夹里新建`hello_yolo.py`
3. 复制下面的代码：

```python
from ultralytics import YOLO
import torch

print("--------------------------------")
print(f"PyTorch 版本: {torch.__version__}")
# 因为你选了CPU版，这里应该会显示 False，这是正常的
print(f"CUDA (显卡加速) 可用: {torch.cuda.is_available()}") 
print("--------------------------------")

print("正在下载并加载 YOLOv8n 模型...")
# 第一次运行会自动从官网下载 yolov8n.pt (约6MB)，请保持联网
model = YOLO('yolov8n.pt')

print("正在检测测试图片...")
# 对一张在线图片进行检测
results = model('https://ultralytics.com/images/bus.jpg')

print("✅ 恭喜！你的环境配置完美，YOLO 运行成功！")
print("--------------------------------")
```

4. 直接在 VS Code 的终端里运行
- 注意：在 VS code中，需要先调整为（yolo）环境
- 步骤如下
  1. 看 **VS Code 的右下角**。
  2. 你应该能看到一个 Python 版本号（比如 `3.x.x`）。
  3. **点击它**，屏幕上方会弹出一个列表。
  4. 在列表里仔细找，找到 **带有 `('yolo': conda)` 字样** 的那一行。
  * *它可能叫 `Python 3.9.x ('yolo': conda)`*。


  1. **选中它**。
  2. 现在再点右上角的“播放/运行”按钮。

**若看到 `✅ 恭喜！` 的字样，和 `image 1/1 ... 4 persons, 1 bus` 的检测结果，那就说明配置成功**
